# Tutorial
### What is git ??
#### Git is a a tool which can be used to check and monitor the current and the previous versions of the code and anytime we can switch between different versions. It is used for version history and we can check when and what changes were made by whome.

### How to push code on github??
#### first using terminal locate the folder which has out files and use the command git init to initialise git . Then add the files into the staging areas using the command git add "file name " or git add . to add all the files. Then use git commit -m "Commit message " to commit the files form th estaging area. open github and copy the URL of the github repo which we have made. use git remote add remote_name url to add new remote. Then write git push remote_name branch_name to push the file on github.
